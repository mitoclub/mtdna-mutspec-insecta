File	egg_dataset.tsv

Date	2018

Description	A tab delimited dataset of insect egg shape and size from the published literature, including additional variables (e.g. volume, aspect ratio) calculated from data in egg_dataset_raw_values.tsv. See publication methods for details on how final values were calculated.

Article	A dataset of egg size and shape from more than 6,700 insect species

Authors	*Church, Samuel H.
	*Donoughe, Seth
	De Medeiros, Bruno A. S.
	Extavour, Cassandra G.
	* equal contribution

Correspondence	church@g.harvard.edu

Column names	ID   :   dataset entry number
	bibtex   :   bibliographic identifier
	author   :   source publication authors
	journal   :   source publication journal
	year   :   source publication year
	family   :   taxonomic family
	superfamily   :   taxonomic superfamily
	subfamily   :   taxonomic subfamily
	suborder   :   taxonomic suborder
	order   :   taxonomic order
	tribe   :   taxonomic tribe
	genus   :   taxonomic genus
	species   :   taxonomic species
	name   :   taxonomic name
	image   :   pdf page number of image
	txtX1   :   text description of egg length (mm)
	txtX2   :   text description of egg width (mm)
	txtvol   :   text description of egg volume (mm3)
	txtar   :   text description of egg aspect ratio
	txtel   :   text description of egg ellipticity
	logtxtar   :   log10 text aspect ratio
	logtxtX1   :   log10 text length
	logtxtX2   :   log10 text width
	logtxtvol   :   log10 text volume
	imX1   :   image measurement length (mm)
	imX2   :   image measurement width (mm)
	logimX1   :   log10 image length
	logimX2   :   log10 image width
	imvol   :   image measurement volume (mm3)
	imar   :   image measurement aspect ratio
	imel   :   image measurement ellipticity
	logimar   :   log10 image aspect ratio
	asym   :   image measurement asymmetry
	curv   :   image measurement curvature (deg)
	sqasym   :   square root image asymmetry
	sqcurv   :   square root image curvature
	X1   :   length (mm)
	X2   :   width (mm)
	X3   :   breadth (mm)
	logX1   :   log10 length
	logX2   :   log10 width
	logX3   :   log10 breadth
	ar   :   aspect ratio
	el   :   ellipticity
	vol   :   volume (mm3)
	logar   :   log10 aspect ratio
	logvol   :   log10 volume
	group   :   taxonomic group
